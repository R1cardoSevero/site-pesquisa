PrestaShop O que é?

Site informativo

O que são?

PrestaShop é um site que oferece uma ferramenta para o desenvolvimento e design de páginas para e-commerce, além de ser um provedor de hospedagem.

Para que servem?

O PrestaShop serve para criarmos nosso site e podermos hospedá-lo de modo que possamos aumentar o escopo de clientes, podendo gerar maior lucro para empresas pequenas que estão começando no mercado com pouca rentabilidade.

Como Funciona?

Ele funciona a partir do site e de um software de Open Source, sendo gratuito para o público. Nele, o usuário pode construir seu site a partir de "peças", sendo de fácil entendimento.

O que posso criar?

Você irá criar seu website para seu e-commerce podendo hospedá-lo em um site online, onde ele será visualizado por diversas pessoas.
